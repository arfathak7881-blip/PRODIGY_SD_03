# Contact Management System (Java)

## 📌 Project Overview

The Contact Management System is a simple Java console application that helps users manage contact information. It provides a menu-driven interface to add, view, update, and delete contact details.

This project is designed for beginners to understand Java programming concepts such as loops, switch statements, user input handling, and menu-driven applications.

---

## ✨ Features

- Add a new contact
- View contact details
- Edit existing contact information
- Delete a contact
- Menu-driven interface
- Easy-to-use console application

---

## 🛠 Technologies Used

- Java
- Scanner Class
- Console Application

---

## 📂 Project Structure

```
ContactManagementSystem.java
```

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/Contact-Management-System-Java.git
```

2. Open the project in VS Code, Eclipse, or IntelliJ IDEA.

3. Compile the program

```bash
javac ContactManagementSystem.java
```

4. Run the program

```bash
java ContactManagementSystem
```
